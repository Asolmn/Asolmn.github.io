使用Baidu-AI进行图像识别

2019-06-13 Python

因为专业课作业，所以有了这个程序，现在基本已经写完。顺便水一遍，整体并没有什么特别有技术含量的地方，基本都是调用API

图像识别-货币

首先是图像识别的代码，要先在百度AI创建应用，然后获取到APP_ID,API_KEY，SECRET_KEY三个值

因为使用的货币识别，目前并没有封装到SDK中，所以我们需要手动发送调用api请求。获取到"access_token"

具体代码如下:

#获取access_token
host = 'https://aip.baidubce.com/oauth/2.0/token?grant_type=client_credentials&client_id='+API_KEY+'&client_secret='+SECRET_KEY
request = urllib.request.Request(host)
request.add_header('Content-Type','application/json;charset=UTF-8')
response = urllib.request.urlopen(request)
content = response.read()
if (content):
	print(content)

定义一个url的变量，变量的值为api的接口地址

requestUrl = 'https://aip.baidubce.com/rest/2.0/image-classify/v1/currency'

读取图片，并且转换为二进制Base64

fileName = str(input('图片名称:'))
with open(fileName,'rb') as f:
    img = base64.b64encode(f.read())

对图片进行转码

params = {'image':img}
params = urllib.parse.urlencode(params).encode('utf-8')

完成这些步骤后，就可以调用api了
前面获取的content中，把access_token提取出来
添加到url上面

requestUrl = requestUrl+"?access_token="+access_token

后面就是发送了，使用python的标准库urllib发送数据

requestUrl = requestUrl+"?access_token="+access_token
req = urllib.request.Request(url=requestUrl,data=params)
req.add_header('Content-Type',"application/x-www-form-urlencoded")
res = urllib.request.urlopen(req)
con = res.read()
